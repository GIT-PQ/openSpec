# PRD：分类历史页面 batchId 筛选增强

## 1. 背景与需求

### 1.1 问题定位

分类历史页面 (`/classification-history`) 缺少对 batchId 的显式筛选功能。

### 1.2 现状

| 功能 | 现状 |
|------|------|
| batchId 筛选输入框 | ❌ 筛选区域无输入框，仅从 URL query 参数获取 |
| batchId 表格列显示 | ❌ 表格无独立列显示 batchId |
| 详情弹窗 batchId 显示 | ✅ 对 batch 类型会显示完整 batchId |
| 后端 batchId 查询支持 | ✅ MyBatis Mapper 已支持等值匹配 |

### 1.3 目标

1. 筛选区域新增 batchId 输入框，支持等值匹配查询
2. 表格新增 batchId 列，完整显示（36字符）

---

## 2. 改动范围

### 2.1 前端改动

| 文件 | 改动点 |
|------|--------|
| `ClassificationHistory.vue` | 筛选区域新增 batchId 输入框 |
| `ClassificationHistory.vue` | 表格新增 batchId 列（完整显示） |
| `ClassificationHistory.vue` | 查询参数构建（已有，无需改动） |
| `ClassificationHistory.vue` | 重置逻辑（已有，无需改动） |

### 2.2 后端改动

无需改动。MyBatis Mapper 已支持 `batchId` 等值匹配查询。

---

## 3. 前端筛选区域设计

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  筛选区域                                                                    │
│                                                                             │
│  [预测类别▼]  [分类时间 起-止]  [来源类型▼]  [摘要关键字___]  [批次ID___]      │
│                                                                             │
│  [查询] [重置] [导出Excel]                                                   │
└─────────────────────────────────────────────────────────────────────────────┘
```

**新增元素**：

| 元素 | 类型 | 说明 |
|------|------|------|
| 批次ID | `el-input` | 普通文本输入，无长度/格式限制，clearable |

**排版建议**：将批次ID放在筛选条件末尾，与摘要关键字同行或独立一行，保持视觉整洁。

---

## 4. 表格列设计

```
┌────┬──────────────────────┬──────────┬──────┬──────────────────────────┬──────┬──────────┬──────┐
│序号│ 摘要                  │预测类别  │置信度│ 批次ID                   │来源  │分类时间  │操作  │
├────┼──────────────────────┼──────────┼──────┼──────────────────────────┼──────┼──────────┼──────┤
│ 1  │ 一种基于...成像...    │医用成像  │85.23%│550e8400-e29b-41d4-a716...│单条  │04-28    │详情  │
│ 2  │ 本发明涉及...手术...  │有源手术  │72.15%│                          │批量  │04-28    │详情  │
└────┴──────────────────────┴──────────┴──────┴──────────────────────────┴──────┴──────────┴──────┘
```

**新增列**：

| 列名 | 字段 | 宽度 | 说明 |
|------|------|------|------|
| 批次ID | `batchId` | 约280px | 完整显示36字符，空值显示空白或 "-" |

**空值处理**：单条分类（source='single'）的 batchId 为空，显示 "-" 或空白。

---

## 5. 验收标准

- 筛选区域新增批次ID输入框，支持输入文本进行等值匹配查询
- 输入批次ID后点击查询，仅显示 batchId 等于输入值的记录
- 表格新增批次ID列，完整显示36字符 UUID
- 单条分类记录的批次ID列显示 "-" 或空白
- 点击重置按钮，批次ID输入框清空
- 前端筛选区域排版整洁，与现有筛选条件协调
