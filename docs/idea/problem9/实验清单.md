## 一、数据集说明

所有消融实验和对比实验均使用**增强后的数据集**。

**理由**：
- 数据增强解决的是数据不平衡问题（数据层面）
- 消融/对比实验考察的是方法本身的效果（模型层面）
- 若不增强，"输血、透析和体外循环器械"类别样本仅31条，所有模型均无法学习该类，对比失去意义
- 论文中可表述为："为保证少数类学习的可行性，所有对比实验均在增强数据集上进行，增强策略详见3.5节"

### 回应专家的逻辑

**论文表述**：
> "微调BERT可视为注入医疗器械领域知识的过程。预训练BERT虽具备通用语义表示能力，但对专利文本中的专业术语（如'超声换能器'、'跨膜压差'）及领域特定语义关系缺乏理解。通过微调第12层Transformer，模型参数向医疗器械语义空间适配，能够更准确地捕捉专业术语的深层含义及其在分类任务中的判别作用。"

**消融结果预期**：
- A1 Macro F1 预期显著低于 A2（论文旧结果：不微调0.5291 vs 微调0.7306）
- 证明：领域知识注入对专业术语密集文本的分类至关重要

## 四、对比模型实验（回应专家意见#1）

> 专家意见：对比基线严重陈旧，需要增加近年专利分类/文本分类优秀方法

### 为什么不能直接引用文献对比？

| 问题 | 说明 |
|------|------|
| 数据集不同 | 各研究使用不同专利数据集，类别体系差异大（多为IPC/CPC分类，本文为《医疗器械分类目录》22类） |
| 类别数不同 | 跨数据集Macro F1不可比（类别数影响计算） |
| 语言不同 | 多数专利分类研究基于英文专利，本文为中文专利 |

**结论**：直接引用文献结果缺乏科学性，必须在**相同数据集**上对比。
