# 抽样推断统计知识

## 1. 核心问题

我们已知d2（1104条，有校对）的错误率，想推断d3（57,000条，无校对）的错误率是否与d2一致。不可能校对全部57,000条，只能抽样。

## 2. 置信区间（Confidence Interval, CI）

### 2.1 直觉

你从d3低置信层抽了330条，发现42条错误，错误率 = 42/330 = 12.73%。

但这个12.73%是**这330条**的错误率，不是d3全部24,236条的真实错误率。换一批330条，可能得到11%或14%。

**置信区间**回答的问题是：基于这330条的观测，d3全部低置信样本的真实错误率大概率在哪个范围？

### 2.2 计算

对于比例 p̂ = 错误数/样本数，95%置信区间为：

```
CI = [p̂ - 1.96 × SE,  p̂ + 1.96 × SE]

其中 SE = √(p̂ × (1-p̂) / n)
```

- SE（标准误）：衡量抽样波动的幅度，n越大SE越小
- 1.96：正态分布95%对应的z值
- 95%的含义：如果重复抽样100次，约95次的CI会包含真实值

### 2.3 例子

| 场景 | n | 错误数 | p̂ | SE | 95% CI |
|------|---|--------|------|------|--------|
| 低置信1轮 | 110 | 14 | 12.73% | 3.19% | [6.59%, 19.09%] |
| 低置信3轮 | 330 | 42 | 12.73% | 1.84% | [9.23%, 16.45%] |
| 低置信5轮 | 550 | 70 | 12.73% | 1.42% | [10.04%, 15.64%] |

**CI宽度 = CI上界 - CI下界**，即"估计的不确定性范围"：
- 1轮：19.09% - 6.59% = **12.5%** → 真实值可能在7%到19%之间，太宽了
- 3轮：16.45% - 9.23% = **7.2%** → 真实值可能在9%到17%之间，可接受
- 5轮：15.64% - 10.04% = **5.6%** → 更精确，但标注量大

**n越大 → SE越小 → CI越窄 → 估计越精确**

## 3. 假设检验

### 3.1 什么是假设检验

我们想知道"d3的错误率是否等于d2的错误率"，这需要假设检验，而不是仅看数字是否接近。

### 3.2 假设设定

- **H0（零假设）**：d3的错误率 = d2的错误率
- **H1（备择假设）**：d3的错误率 ≠ d2的错误率

### 3.3 检验逻辑

假设H0成立（d3错误率确实等于d2），那么我们观测到的d3抽样错误率应该围绕d2的错误率波动。如果波动在正常范围内 → 不能拒绝H0；如果偏离太远 → 拒绝H0。

**z统计量**：

```
z = (p̂_d3 - p_d2) / SE0

其中 SE0 = √(p_d2 × (1-p_d2) / n)
```

- SE0用H0下的错误率（即d2的值）计算，而非观测值
- z的含义：观测偏差是标准误的几倍

**p-value**：

```
p-value = 2 × (1 - Φ(|z|))
```

其中Φ是标准正态分布的累积分布函数。

| z绝对值 | p-value | 含义 |
|---------|---------|------|
| < 1.96 | > 0.05 | 偏差不显著，不拒绝H0 |
| ≥ 1.96 | ≤ 0.05 | 偏差显著，拒绝H0 |
| ≥ 2.58 | ≤ 0.01 | 偏差高度显著 |

### 3.4 例子

d2低置信错误率 = 12.84%。假设从d3低置信抽330条，观测错误率12.73%：

```
SE0 = √(0.1284 × 0.8716 / 330) = 0.01839
z = (0.1273 - 0.1284) / 0.01839 = -0.060
p-value = 2 × (1 - Φ(0.060)) = 0.952
```

p-value = 0.952 >> 0.05，**不拒绝H0**，即d3与d2错误率一致。

## 4. 关键问题：观测到多少错误率才会拒绝H0？

即"我每次抽样错误率在什么范围内，才能不拒绝H0？"

### 4.1 拒绝域

对双侧检验（α=0.05），拒绝H0的条件：

```
|z| ≥ 1.96
即 |p̂ - p_d2| ≥ 1.96 × SE0
```

所以**不拒绝H0的观测错误率范围**：

```
p_d2 - 1.96 × SE0  ≤  p̂  ≤  p_d2 + 1.96 × SE0
```

### 4.2 不同轮次下的不拒绝范围

以低置信层（d2错误率=12.84%）为例：

| 轮次 | n | SE0 | 不拒绝H0的错误率范围 | 不拒绝H0的准确率范围 |
|------|---|-----|--------------------|--------------------|
| 1轮 | 110 | 3.19% | [6.59%, 19.09%] | [80.91%, 93.41%] |
| 2轮 | 220 | 2.26% | [8.42%, 17.26%] | [82.74%, 91.58%] |
| 3轮 | 330 | 1.84% | [9.23%, 16.45%] | [83.55%, 90.77%] |
| 5轮 | 550 | 1.43% | [10.04%, 15.64%] | [84.36%, 89.96%] |

以高置信层（d2错误率=2.09%）为例：

| 轮次 | n | SE0 | 不拒绝H0的错误率范围 | 不拒绝H0的准确率范围 |
|------|---|-----|--------------------|--------------------|
| 1轮 | 110 | 1.36% | [0.00%, 4.76%] | [95.24%, 100%] |
| 2轮 | 220 | 0.96% | [0.20%, 3.98%] | [96.02%, 99.80%] |
| 3轮 | 330 | 0.79% | [0.55%, 3.63%] | [96.37%, 99.45%] |
| 5轮 | 550 | 0.61% | [0.89%, 3.29%] | [96.71%, 99.11%] |

**解读**：
- n越大，不拒绝范围越窄 → 对d3的质量要求越严格
- 3轮低置信：准确率在83.6%~90.8%之间就不拒绝H0
- 3轮高置信：准确率在96.4%~99.5%之间就不拒绝H0

### 4.3 整体准确率的不拒绝范围

综合高/低置信层（加权57.8%/42.2%）：

| 轮次 | 整体准确率不拒绝范围 | d2基准 |
|------|--------------------|----|
| 1轮 | [86.3%, 96.0%] | 93.4% |
| 2轮 | [88.1%, 95.3%] | 93.4% |
| 3轮 | [89.0%, 94.7%] | 93.4% |
| 5轮 | [89.7%, 94.2%] | 93.4% |

**3轮时**：d3抽样整体准确率在89%~95%之间，就认为d3与d2一致。

## 5. 如何确定抽样轮次

### 5.1 取决于你想要多"严格"

| 需求 | 推荐轮次 | 总校对量 | 理由 |
|------|----------|----------|------|
| 初步验证，时间紧 | 2轮 | 440条 | CI较宽但能说明大致趋势 |
| 论文可引用，有统计说服力 | 3轮 | 660条 | CI宽度适中，检验有区分度 |
| 高精度，审稿人可能追问 | 5轮 | 1100条 | CI窄，结果稳健 |

### 5.2 实际考量

- 高置信层错误率仅2%，1轮110条期望只出现2个错误，单轮估计极不稳定
- 3轮330条期望7个错误，足以让CI下界为正（即能排除"0错误"的极端情况）
- 低置信层错误率13%，3轮330条期望42个错误，估计已经比较稳定

### 5.3 建议

**3轮**。理由：
1. 高置信层期望错误数>5，正态近似成立
2. 低置信层CI宽度~7%，精度可接受
3. 总标注量660条，1-2天可完成
4. 论文中可以报告CI和p-value，统计规范
