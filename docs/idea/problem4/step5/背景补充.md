# 背景
- 数据D1，58000左右，22类，但原始的标签质量低。因此，计算了D1每个样本的类内相似度，通过分层抽样得到了D2。
- D2，1104条，22类，并进行了专家标注。
- 然后用JS散度量化了D1和D2类内相似度分布的差异，22类均小于0.1，说明D2的分布与D1非常接近，可以代表D1的整体分布。
- D3数据集的定义为：D3 = D1 - D2，即D1中未被抽取到D2的样本组成的集合，57000左右。
- 现在我构建了一个标注工作流，用D2作为评估集，在D2上得到了0.9321的acc。
- 但是盲审专家认为：标签纠错、错误检测、置信度过滤等处理也没有考虑。可能是质疑我D3数据集的质量。由此引出问题。
