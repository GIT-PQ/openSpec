# 医疗器械分类标准辅助一级产品分类分析

## 背景

目标：按照医疗器械分类目录，将医疗器械专利映射到22个一级产品类别（非监管类别）。

22类一级产品类别：

> LABELS = ["有源手术器械", "无源手术器械", "神经和心血管手术器械", "骨科手术器械", "放射治疗器械", "医用成像器械", "医用诊察和监护器械", "呼吸、麻醉和急救器械", "物理治疗器械", "输血、透析和体外循环器械", "医疗器械消毒灭菌器械", "有源植入器械", "无源植入器械", "注输、护理和防护器械", "患者承载器械", "眼科器械", "口腔科器械", "妇产科、辅助生殖和避孕器械", "医用康复器械", "中医器械", "医用软件", "临床检验器械"]

以下从《医疗器械分类规则》（2015年版）中选取可作为LLM系统提示词辅助一级产品分类的原句，并逐一说明理由。

---

## 1. 有源/无源的定义（区分4个类别的核心分界线）

**原文：**

> 第三条（二）无源医疗器械：不依靠电能或者其他能源，但是可以通过由人体或者重力产生的能量，发挥其功能的医疗器械。
>
> 第三条（三）有源医疗器械：任何依靠电能或者其他能源，而不是直接由人体或者重力产生的能量，发挥其功能的医疗器械。

**理由：** 22类中有4类直接以有源/无源命名——"有源手术器械"vs"无源手术器械"、"有源植入器械"vs"无源植入器械"。专利文本通常会描述器械的能源来源（电驱动、超声、激光等 vs 手动、机械），这两条定义让LLM能首先判断有源/无源属性，直接排除一半候选类别。这是分类树的第一层分叉。

---

## 2. 植入器械的定义（区分植入类与手术类）

**原文：**

> 第三条（六）植入器械：借助手术全部或者部分进入人体内或腔道（口）中，或者用于替代人体上皮表面或眼表面，并且在手术过程结束后留在人体内30日（含）以上或者被人体吸收的医疗器械。

**理由：** 专利中是否描述"植入后长期留存"或"可吸收"是区分"有源植入器械/无源植入器械"与"有源手术器械/无源手术器械"的关键。手术器械是术中使用、术后取出；植入器械是术后留存在体内。30日这个时间阈值和"被人体吸收"是明确的判断依据。

---

## 3. 重复使用手术器械的定义（含动作动词，直接对应无源手术器械）

**原文：**

> 第三条（五）重复使用手术器械：用于手术中进行切、割、钻、锯、抓、刮、钳、抽、夹等过程，不连接任何有源医疗器械，通过一定的处理可以重新使用的无源医疗器械。

**理由：** 这9个动作动词（切、割、钻、锯、抓、刮、钳、抽、夹）是专利文本中高频出现的功能性描述词。LLM在专利摘要中识别到"钳夹""切割""钻入"等动作时，可直接关联到"无源手术器械"类别。同时"不连接任何有源医疗器械"进一步强化了有源/无源的区分。

---

## 4. 侵入器械的定义（区分手术类与非手术接触类）

**原文：**

> 第三条（四）侵入器械：借助手术全部或者部分通过体表侵入人体，接触体内组织、血液循环系统、中枢神经系统等部位的医疗器械，包括介入手术中使用的器材、一次性使用无菌手术器械和暂时或短期留在人体内的器械等。本规则中的侵入器械不包括重复使用手术器械。

**理由：** "侵入"vs"植入"vs"非侵入"是区分手术器械（有源/无源手术器械）、植入器械、注输护理器械等的关键。"借助手术通过体表侵入"但"不长期留存"的特征指向手术器械类别。末句"不包括重复使用手术器械"也提示LLM：重复使用的手术工具归无源手术器械，而非笼统的侵入器械。

---

## 5. 使用形式分类表（与22类直接映射的顶层结构）

**原文：**

> 第五条（三）根据不同的结构特征和是否接触人体，医疗器械的使用形式包括：
>
> - 无源接触人体器械：液体输送器械、改变血液体液器械、医用敷料、侵入器械、重复使用手术器械、植入器械、避孕和计划生育器械、其他无源接触人体器械。
> - 无源非接触人体器械：护理器械、医疗器械清洗消毒器械、其他无源非接触人体器械。
> - 有源接触人体器械：能量治疗器械、诊断监护器械、液体输送器械、电离辐射器械、植入器械、其他有源接触人体器械。
> - 有源非接触人体器械：临床检验仪器设备、独立软件、医疗器械消毒灭菌设备、其他有源非接触人体器械。

**理由：** 这四组使用形式与22个一级产品类别有直接的语义映射关系：

| 使用形式（第五条） | 对应一级产品类别 |
|---|---|
| 液体输送器械 | 输血、透析和体外循环器械 |
| 能量治疗器械 | 有源手术器械、物理治疗器械、放射治疗器械 |
| 诊断监护器械 | 医用诊察和监护器械、医用成像器械 |
| 电离辐射器械 | 放射治疗器械 |
| 临床检验仪器设备 | 临床检验器械 |
| 避孕和计划生育器械 | 妇产科、辅助生殖和避孕器械 |
| 医疗器械清洗消毒器械 | 医疗器械消毒灭菌器械 |

这四组列表相当于给LLM提供了一个2×2的分类框架（有源/无源 × 接触/非接触人体），让LLM先定位大类，再在对应象限内细分。

---

## 6. 独立软件的定义（直接对应"医用软件"类别）

**原文：**

> 第三条（十五）独立软件：具有一个或者多个医疗目的，无需医疗器械硬件即可完成自身预期目的，运行于通用计算平台的软件。

**理由：** "医用软件"是22类中唯一一个纯软件类别。专利中如果描述的是算法、数据处理、AI诊断等方法，且不依赖特定硬件，LLM可据此将其归入"医用软件"而非"医用诊察和监护器械"或"临床检验器械"。"无需医疗器械硬件"和"通用计算平台"是关键的区分条件。

---

## 7. 解剖部位定义（区分专科器械类别）

**原文：**

> 第三条（十）腔道（口）：口腔、鼻腔、食道、外耳道、直肠、阴道、尿道等人体自然腔道和永久性人造开口。
>
> 第三条（十二）组织：人体体内组织，包括骨、牙髓或者牙本质，不包括血液循环系统和中枢神经系统。
>
> 第三条（十三）血液循环系统：血管（毛细血管除外）和心脏。
>
> 第三条（十四）中枢神经系统：脑和脊髓。

**理由：** 22类中有多个按解剖/专科领域划分的类别：眼科器械、口腔科器械、妇产科器械、骨科手术器械、神经和心血管手术器械。这些定义让LLM能根据专利涉及的解剖部位进行归类：

| 解剖关键词 | 对应一级产品类别 |
|---|---|
| 骨 | 骨科手术器械 |
| 口腔 | 口腔科器械 |
| 血管和心脏 | 神经和心血管手术器械 |
| 阴道 | 妇产科、辅助生殖和避孕器械 |
| 脑和脊髓 | 神经和心血管手术器械 |

特别值得注意的是"组织"定义中明确"包括骨"，这让涉及骨科的专利不会被误归到一般外科。

---

## 8. 具有计量测试功能的定义（区分检验/监护类与治疗类）

**原文：**

> 第三条（十六）具有计量测试功能的医疗器械：用于测定生理、病理、解剖参数，或者定量测定进出人体的能量或物质的医疗器械，其测量结果需要精确定量，并且该结果的准确性会对患者的健康和安全产生明显影响。

**理由：** "定量测定""精确定量"等特征帮助LLM区分临床检验器械/医用诊察和监护器械（测量为主）与治疗类器械（治疗为主）。专利如果核心功能是"测定参数"而非"施加治疗"，应归入检验或监护类。

---

## 推荐提示词组合策略

按分类决策树顺序排列，构成LLM系统提示词：

1. **第五条（三）的四组使用形式** → 给LLM全局框架（2×2矩阵）
2. **第三条（二）（三）有源/无源定义** → 第一层分叉
3. **第三条（六）植入器械** → 第二层分叉（是否植入）
4. **第三条（四）侵入器械 + （五）重复使用手术器械** → 手术类细分
5. **第三条（十）（十二）（十三）（十四）解剖定义** → 专科类定向
6. **第三条（十五）独立软件** → 医用软件识别
7. **第三条（十六）计量测试功能** → 检验/监护与治疗的区分

## 未选入的条款及原因

| 条款 | 不选原因 |
|---|---|
| 第三条（一）预期目的 | 过于宽泛，对所有类别均适用，无区分力 |
| 第三条（七）接触人体器械 | 与第五条（三）重复，第五条更具操作性 |
| 第三条（八）使用时限（暂时/短期/长期） | 使用时限对监管分类重要，但对一级产品分类影响较小——产品类别不看使用时长 |
| 第三条（九）皮肤、（十一）创伤 | 单独使用区分度不足，需结合其他定义 |
| 第三条（十七）慢性创面 | 仅影响医用敷料的监管类别升级，不影响一级产品分类 |
| 第六条各特殊分类规则 | 属于监管类别升级规则（如药械组合按第三类管理），与产品类别无关，反而可能误导LLM将产品分类与监管分类混淆 |
| 第四条风险程度分级 | 一级产品类别不以风险程度划分，该条指向的是监管类别（Ⅰ/Ⅱ/Ⅲ），与目标任务无关 |
