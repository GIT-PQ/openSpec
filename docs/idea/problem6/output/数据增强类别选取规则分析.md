## 结论

**仅对输血、透析和体外循环器械进行LLM数据增强，其他类别不增强。**

## 推理链路

### Step 1：实验证据——31条样本的类别无法学习

从不使用增强数据的训练log（见`problem6/input/不使用增强数据.log`）中，提取输血透析体外循环类别的测试集recall演变：

| 训练轮次 | 测试集recall | 测试集f1-score | 状态 |
|---------|-------------|---------------|------|
| 第1轮 | 0.00 | 0.00 | 完全失效 |
| 第2轮 | 0.00 | 0.00 | 完全失效 |
| 第3轮 | 0.00 | 0.00 | 完全失效 |
| 第4轮 | 0.00 | 0.00 | 完全失效 |
| 第5轮 | 0.00 | 0.00 | 完全失效 |
| 第6轮 | 0.00 | 0.00 | 完全失效 |
| 第7轮 | 0.00 | 0.00 | 完全失效 |
| 第8轮 | 0.00 | 0.00 | 完全失效 |
| 第9轮 | 0.33 | 0.50 | 勉强学习 |

**关键事实**：原始31条样本，按8:2划分后训练集仅约25条、测试集仅3条。模型在10轮训练中，前8轮对该类的recall始终为0，第9轮才勉强达到0.33。这表明**31条样本不足以让模型学习该类别的分类边界**。

### Step 2：对比证据——98条样本的类别可以正常学习

同样从不使用增强数据的log中，提取样本量最接近的医疗器械消毒灭菌器械（98条）的表现：

| 训练轮次 | 测试集recall | 测试集f1-score |
|---------|-------------|---------------|
| 第1轮 | 0.44 | 0.62 |
| 第2轮 | **0.89** | 0.94 |
| 第3轮+ | 0.78-1.00 | 0.87-1.00 |

**关键事实**：98条样本从第2轮开始recall就稳定在0.89以上，说明模型已能有效学习该类别。

**31条与98条之间存在质的断裂**：前者10轮训练中8轮完全失效，后者2轮训练即稳定学习。

### Step 3：增强效果验证

从使用增强数据的训练log（见`problem6/input/使用增强数据.log`）中，输血透析体外循环类别的表现演变：

| 训练轮次 | 测试集recall | 测试集f1-score | 状态 |
|---------|-------------|---------------|------|
| 第1轮 | 0.00 | 0.00 | 未学习 |
| 第2轮 | 0.22 | 0.36 | 开始学习 |
| 第3轮 | 0.56 | 0.71 | 快速提升 |
| 第4轮 | 0.67 | 0.80 | 稳定提升 |
| 第5轮 | 0.67 | 0.80 | |
| 第6轮 | 0.78 | 0.82 | |
| 第7轮 | 0.89 | 0.89 | |
| 第8轮 | 0.78 | 0.88 | |
| 第9轮 | **0.89** | **0.94** | 达到可用水平 |

**关键事实**：增强后（31→93条），模型从第2轮开始就能学习该类别，recall从0→0.22→0.89，最终达到与98条类别相当的水平。

### Step 4：其他少量类别的表现分析

从实验数据看，其他样本量较少的类别在不增强情况下也能学习：

| 类别 | 原始样本 | 测试集样本 | 最终recall(不增强) | 最终recall(增强) |
|------|---------|-----------|------------------|----------------|
| 医疗器械消毒灭菌器械 | 98 | 9 | 1.00 | 1.00 |
| 患者承载器械 | 107 | 10 | 1.00 | 0.90 |
| 有源植入器械 | 193 | 19 | 0.84 | 0.68 |
| 呼吸麻醉急救器械 | 220 | 22 | 0.95 | 0.73 |

这些类别在不增强时已能达到较高recall，增强后反而部分类别表现下降（可能因增强数据引入噪声或改变了数据分布）。**说明它们已跨过最低学习门槛，增强的收益不确定甚至可能为负**。

注：消毒灭菌器械类别虽样本仅98条，但类别特征明显（涉及清洁消毒的术语易区分），因此模型能快速学习。

### Step 5：结论——仅对无法学习的类别增强

综合实验证据：
1. **31条的类别无法学习**（10轮训练8轮recall=0）
2. **98条及以上的类别可以学习**（第2轮即稳定）
3. **增强使31条类别从无法学习→可以学习**（recall 0→0.89）
4. **增强对98条及以上类别的收益不确定**（部分类别反而下降）

因此，**仅对输血透析体外循环器械进行增强**，其他类别不需要增强。

---

## 选取规则的表述

> **规则：对在训练中无法学习的类别进行LLM数据增强。判断标准为：若某类别在常规训练过程中测试集recall持续为0或接近0，则判定为无法学习，需进行增强。**

当前仅输血、透析和体外循环器械满足此条件。
