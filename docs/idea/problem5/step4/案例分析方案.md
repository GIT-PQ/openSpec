# 案例分析方案

## 选取原则

选取3条样本，每条承担独立的论证功能，互不重叠，形成完整论证闭环。

| 案例 | 论证功能 | 回答审稿人的什么问题 |
|------|----------|---------------------|
| 逐级修正型 | 方法各组件的**递进贡献** | "你的方法比其他标注方法好在哪里？" |
| 纠错修正型 | 纠错模块的**独立价值** | "纠错环节是否真的有必要？去掉RAG已经很好了" |
| 失败案例型 | 方法的**边界与局限** | "你的方法在什么情况下会失败？" |

> 不再单独设置"原始标签错误修正型"案例。原始标签acc(0.5833) vs 本文方法acc(0.9321)的定量对比已充分说明问题，逐级修正型案例中天然包含"原始标签错→本文方法对"的对比，无需重复。

## 案例1：逐级修正型

> 筛选条件：Keyword错 ∧ Zero-Shot错 ∧ Prompt错 ∧ RAG对 ∧ New对
> 论证目标：展示方法各组件的递进贡献，一条样本串起所有方法

### 备选1A（推荐）：CN200910105446.X — "肝脏纤维化检测装置"

**摘要**：

> 本发明提供了一种肝脏纤维化检测装置，用以检测体内肝脏的纤维化程度，其包括超声探头、与该超声探头电连接的探头跟踪系统、与该超声探头电连接的信号处理模块、以及与该探头跟踪系统和信号处理模块的且带有系统软件的计算机，该超声探头向体内发射低频振动，同时利用超声探测该振动所形成的剪切波在肝脏组织内传播，该探头跟踪系统构建空间三维坐标系，并实时检测超声探头相对于肝组织的位置和角度，该信号处理模块采集并处理探头获得的回波电信号，通过计算机处理分析获得相应坐标和角度下肝脏纤维化的实时定量信息；本发明解决了现有技术有创、费用高以及不能从各个方位和角度评估肝脏组织硬度的问题。

**各方法预测结果**：

| 方法 | 预测标签 | 正确 |
|------|----------|------|
| 原始标签 | 医用成像器械 | 错 |
| Keyword | 医用成像器械 | 错 |
| LLM-Zero-Shot | 医用成像器械 | 错 |
| LLM-Prompt | 医用成像器械 | 错 |
| KNN | 医用诊察和监护器械 | 对 |
| LLM-Prompt-RAG | 医用诊察和监护器械 | 对 |
| Old | 医用诊察和监护器械 | 对 |
| **New（本文方法）** | **医用诊察和监护器械** | **对** |

**分析要点**：

- **Keyword/LLM-Zero-Shot/LLM-Prompt均错**：三种方法均归为"医用成像器械"。摘要中"超声探头""回波电信号"等词在语义上与超声成像高度关联，但这台设备的本质功能是"检测肝脏纤维化程度"——通过剪切波速度定量评估组织硬度，属于诊察/监护（实时定量检测生理指标），而非成像（获取解剖结构图像）。
- **LLM-Prompt-RAG对**：RAG检索到"医用诊察和监护器械"下"组织硬度检测""纤维化评估"等条目，与摘要中"检测纤维化程度""实时定量信息"精确匹配，纠正了LLM的误判。
- **原始标签错**：原始标注也将其归为"医用成像器械"，说明此混淆具有普遍性——使用超声技术的设备容易被默认归为成像。
- **混淆点**：医用成像器械 vs 医用诊察和监护器械（分类规则："影像是宏观结构" vs "患者实时监测"），该设备获取的是定量指标而非结构图像。
- **优势**：标题"肝脏纤维化检测装置"是明确的医疗器械；混淆点（成像 vs 诊察监护）是分类规则中明确列出的区分重点；原始标签也错，增强论证力度。

## 案例2：纠错修正型

> 筛选条件：LLM-Prompt-RAG错 ∧ New对
> 论证目标：证明纠错模块（条件分支子LLM）的独立价值

### 备选2A（推荐）：CN202011051118.9 — "一种结合药物递送的深部柔性脑电极的植入方法"

**摘要**：

> 本发明公开了一种结合药物递送的深部柔性脑电极的植入方法，所述植入方法包括柔性脑电极预处理步骤和柔性脑电极植入步骤，所述柔性脑电极预处理步骤具体包括：在所述柔性脑电极的表面形成固化的蛋白保护层，其中，所述固化的蛋白保护层中包括待递送的药物。本发明通过在柔性脑电极的表面形成一层固化的蛋白保护层，其杨氏模量能够提升到足以植入脑内的程度，具有良好的生物相容性和微创性；蛋白保护层内包含一定浓度的待递送的药物，能够有效抑制术中感染和术后炎症反应；本发明的柔性脑电极的植入操作较为便捷简单，植入成功率高。

**各方法预测结果**：

| 方法 | 预测标签 | 正确 |
|------|----------|------|
| 原始标签 | 有源植入器械 | 对 |
| Keyword | 物理治疗器械 | 错 |
| LLM-Zero-Shot | 无源植入器械 | 错 |
| LLM-Prompt | 有源植入器械 | 对 |
| KNN | 无源植入器械 | 错 |
| LLM-Prompt-RAG | 无源植入器械 | 错 |
| Old | 有源植入器械 | 对 |
| **New（本文方法）** | **有源植入器械** | **对** |

**分析要点**：

- **LLM-Prompt-RAG错**：RAG检索到的目录条目可能包含"植入""柔性材料"等指向"无源植入器械"的内容。摘要中"柔性脑电极"的"柔性"一词让LLM联想到无源材料，忽略了"电极"在分类规则中明确属于"有源植入器械"（规则："电极一般属于有源植入器械"）。
- **New对（纠错修正）**：主LLM输出"无源植入器械"后，触发了"无源植入器械 vs 有源植入器械"的纠错子LLM。该子LLM的提示词明确指出"与电极、电极植入方法或植入式传感器相关，归入有源植入器械"，子LLM识别到摘要中"脑电极""植入方法"等关键词，将结果修正为"有源植入器械"。
- **优势**：摘要中"脑电极""植入"是典型的医疗器械术语，不存在"是否属于医疗器械"的争议；纠错分支（无源植入 vs 有源植入）直接对应工作流中的条件分支，机制解释清晰。

## 案例3：失败案例型

> 筛选条件：New错
> 论证目标：坦诚展示方法局限，失败原因必须可解释

### 备选3A（推荐）：CN202110898105.3 — "一种肝胆外科临床用取石装置"

**摘要**：

> 本发明涉及医疗器械技术领域，公开了一种肝胆外科临床用取石装置，包括箱体、侧板、半圆体、结石放置槽、第一通道、第一操作杆、按压把、第二通道、第二操作杆、把手、第一弧形槽、第二弧形槽、第三弧形槽、第四弧形槽、第一弧形板、第一滑块、第二弧形板、第二滑块、上抓片、第三滑块、下抓片、第四滑块、上挤压板、第一弹性铰链、下挤压板、第二弹性铰链、第一连杆、第二连杆、第三连杆和第四连杆。本装置使用简单，创新性强，使用本装置在患者体内夹取结石后不用取出装置拿取结石就可进行下一次夹取，通过设置碎石电极击碎结石，使结石放置槽内可容纳多颗结石。

**各方法预测结果**：

| 方法 | 预测标签 | 正确 |
|------|----------|------|
| 原始标签 | 无源手术器械 | 错 |
| Keyword | 有源手术器械 | 对 |
| LLM-Zero-Shot | 神经和心血管手术器械 | 错 |
| LLM-Prompt | 无源植入器械 | 错 |
| KNN | 骨科手术器械 | 错 |
| LLM-Prompt-RAG | 无源手术器械 | 错 |
| Old | 无源手术器械 | 错 |
| **New（本文方法）** | **无源手术器械** | **错** |

**分析要点**：

- 正确类别为"有源手术器械"——摘要末尾提及"碎石电极击碎结石"，"电极"暗示有能源驱动，属于有源手术器械。
- **所有方法（除Keyword外）均错**：摘要主体描述了大量机械结构（滑块、弧形板、弹性铰链等），这些描述占据了摘要绝大部分篇幅，强烈指向"无源手术器械"。只有末尾一句"碎石电极"透露了有源特征。
- **New错**：主LLM被大量机械结构描述主导，输出"无源手术器械"。虽然"无源植入器械 vs 有源植入器械"的纠错分支被触发时可能关注"电极"关键词，但主LLM输出的是"无源手术器械"，不触发该分支（条件分支仅对4个特定类别触发），因此纠错机制未被激活。
- **失败原因**：关键分类信息（"电极"）被淹没在大量无关机械描述中，且主LLM的输出不在纠错分支的触发范围内，纠错机制无法介入。
- **优势**：标题"肝胆外科临床用取石装置"是明确的医疗器械；混淆点（有源 vs 无源手术器械）是分类目录的经典难题；失败原因可解释——关键信息占比极低且不在纠错触发范围内。
