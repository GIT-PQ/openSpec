## 输出文件规范

实验产出两个文件，通过 `patent_id` 关联：

### 1. 嵌入数据集（机器用）

- **格式：** parquet（跨平台，列式压缩）
- **路径：** `step2/output/patent_embeddings.parquet`
- **内容：**

| 列名 | 类型 | 说明 |
|------|------|------|
| 申请号 | str | 专利唯一标识 |
| 标题 | str | 专利标题 |
| 摘要 | str | 专利摘要 |
| 一级产品类别（校对） | str | 真实标签（专家标注） |
| 一级产品类别（原始） | str | 专利原始标签 |
| bm25_label | str | BM25预测标签 |
| llm_zs_label | str | LLM zero-shot预测标签 |
| knn_bge_label | str | KNN+BGE预测标签 |
| rag_label | str | RAG+规则方法预测标签 |
| bge | list[float] | BGE嵌入向量（1024维） |

- **用途：** KNN计算、指标评估，代码自包含

### 2. 标注对比表（人可读）

- **格式：** xlsx
- **路径：** `step2/input/D2.xlsx`（追加预测列）
- **内容：**

| 列名 | 说明 |
|------|------|
| 申请号 | 专利唯一标识 |
| 标题 | 专利标题 |
| 摘要 | 专利摘要 |
| 一级产品类别（校对） | 真实标签（专家标注） |
| 一级产品类别（原始） | 专利原始标签 |
| bm25 | BM25预测标签 |
| llm_zero_shot | LLM zero-shot预测标签 |
| knn_bge | KNN+BGE预测标签 |
| rag | RAG+规则方法预测标签 |

- **用途：** Excel中直接对比、筛选、案例分析；论文中挑典型案例

### 3. 分类目录向量库（方法③专用）

- **格式：** parquet
- **路径：** `step2/output/医疗器械分类目录_eb_with_full_structure_normalized.parquet`
- **内容：**

| 列名 | 类型 | 说明 |
|------|------|------|
| 完整结构 | str | 拼接后的分类目录条目（三级序号/产品类别+产品描述+预期用途） |
| 一级序号/产品类别 | str | 类别名称（本次研究的标签） |
| 完整结构_eb_归一化 | list[float] | BGE嵌入向量（1024维，归一化） |
